KnowMe

Know Your System, Instantly.

A fast, offline, and beautifully designed command-line tool for displaying detailed system information.

View on GitHub

Why KnowMe?

Blazing Fast

Get your system information in a blink of an eye. No more waiting for slow, bloated apps.

Fully Offline

Works anywhere, anytime. No internet connection required to get your system's stats.

Beautifully Crafted

Presents information in a clean, colorful, and easy-to-read format with OS-specific ASCII art.

Features

Cross-Platform

Works seamlessly on Windows, macOS, and a wide variety of Linux distributions.

Comprehensive Info

From CPU and RAM to network details and running processes, get all the info you need.

Lightweight

With minimal dependencies, KnowMe is light on resources and won't slow down your system.

System Information Displayed

  • Operating System
  • Kernel Version
  • CPU Model & Cores
  • Memory (RAM) Usage
  • Disk Usage
  • GPU Information
  • Screen Resolution
  • System Uptime
  • IP & Mac Address
  • And much more...

Installation

Quick Install (Recommended)

curl -fsSL https://raw.githubusercontent.com/mehtahrishi/knowme/main/install.sh | sudo bash

APT-style Install

curl -fsSL https://raw.githubusercontent.com/mehtahrishi/knowme/main/apt-install.sh | sudo bash

Direct Download

wget https://github.com/mehtahrishi/knowme/raw/main/deb/knowme_1.0-1_all.deb
sudo dpkg -i knowme_1.0-1_all.deb

Usage

Simply run the command:

knowme

Removal

Quick Install Removal

sudo uninstall-knowme

APT-style Removal

sudo apt remove knowme
sudo apt autoremove

Direct Download Removal

sudo dpkg -r knowme
sudo apt autoremove
