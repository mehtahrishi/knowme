KnowMe APT Repository

Fast, offline system information tool with ASCII art

Installation

Add repository:

echo "deb [trusted=yes] https://mehtahrishi.github.io/knowme/apt stable main" | sudo tee /etc/apt/sources.list.d/knowme.list
sudo apt update

Install:

sudo apt install knowme

Other Installation Methods

  • PyPI: pip install knowme
  • Snap: sudo snap install knowme --classic
  • GitHub: Download .deb from releases
